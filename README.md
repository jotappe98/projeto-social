# Projeto Social – Simulação de Perfis em Smartphone

Este é um projeto front-end desenvolvido exclusivamente com HTML e CSS, cujo objetivo é simular a navegação entre perfis de redes sociais dentro da tela de um smartphone.

Ao clicar nos botões localizados no lado direito da página, os perfis correspondentes são carregados dentro da tela do celular, que é implementada utilizando um iframe.

---

## Contexto de Aprendizado

Este projeto foi desenvolvido como parte dos estudos do Módulo 4 do curso de HTML e CSS do professor Gustavo Guanabara, disponível na plataforma Curso em Vídeo.

Embora inspirado nos conceitos apresentados ao longo do curso, a implementação foi realizada de forma independente, como exercício prático de fixação e experimentação, antes da visualização das aulas em que o projeto é desenvolvido e finalizado pelo professor.

---

## Conceito do Projeto

- Simulação visual de um smartphone em ambiente web
- Navegação entre perfis sociais sem uso de JavaScript
- Conteúdo exibido dinamicamente dentro de um iframe
- Foco em layout, posicionamento e organização utilizando apenas CSS

---

## Tecnologias Utilizadas

- HTML5
- CSS3
- iframe para carregamento dos perfis

Não foram utilizados:
- JavaScript
- Frameworks ou bibliotecas externas

---

## Funcionalidades

- Botões laterais representando diferentes redes sociais
- Cada botão carrega um perfil distinto dentro do iframe
- Perfis com opções visuais como:
  - Seguir
  - Conectar
  - Enviar mensagem
- Layout centralizado simulando a tela de um smartphone

---

## Objetivo

O principal objetivo deste projeto é praticar e consolidar conhecimentos em HTML e CSS, explorando:

- Estruturação semântica
- Posicionamento e alinhamento avançado com CSS
- Uso de iframe para simular navegação interna
- Desenvolvimento de interfaces sem dependência de JavaScript
